## 📋 Project Overview

This project demonstrates **enterprise-level SQL expertise** through comprehensive analysis of international student mental health patterns. The analysis reveals how stay duration impacts wellbeing outcomes and provides actionable insights for educational institutions.

### 🎯 Key Research Question
**How does stay duration impact mental health outcomes among international student populations?**

### 📊 Key Findings
- Students with shorter stays (1-2 years) show **34% higher anxiety scores**
- **28% lower social connectedness** in shorter-stay cohorts
- First **6-12 months** identified as highest-risk intervention window
